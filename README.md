nonebot-plugin-pixivbot-onebot-v11
=====

Onebot V11协议版本的PixivBot

## 配置

额外配置：
```
# 戳一戳功能，可选值：空, ranking, random_recommended_illust, random_bookmark
pixiv_poke_action=random_recommended_illust
```
